# ForgeX™ - Sistema de Personalização para MTA:SA

## Introdução
ForgeX™ é um sistema avançado de personalização para armas e veículos no Multi Theft Auto: San Andreas, projetado para oferecer uma experiência única de modificação.

## Funcionalidades
- **Mods Visuais e Funcionais**
- **Crafting e Evolução**
- **Sistema de Licença**
- **Logs Avançados**
- **Configuração Extensível**

## Instalação
1. Coloque a pasta `ForgeX` no diretório de resources.
2. Configure o arquivo `config/config.json`.
3. Certifique-se de adicionar o resource na ACL.

## Licença
Edite o arquivo `server/license.lua` com o IP autorizado e a data de validade.
